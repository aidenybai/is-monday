# is-monday
[![Version](https://img.shields.io/npm/v/is-monday.svg?maxAge=3600)](https://www.npmjs.com/package/is-monday)

Checks if it is monday (the worst day)

## Install

```bash
npm i -s is-monday
```

## Usage

```js
const monday = require('is-monday');

monday(); // returns boolean.
```
